# Interaction Probing (Axis 2)

This directory holds verb-spatial binding experiments on generative diffusion models. Each subdirectory is a self-contained experiment.

## Experiments

| Experiment | Status | Folder |
|---|---|---|
| AGD20K cross-attention probing | in progress | [`agd20k_cross_attention/`](agd20k_cross_attention/) |

The AGD20K experiment probes whether language-conditioned image/video diffusion models develop verb-anchored cross-attention. Two systems compared: FLUX.1-schnell (text-to-image) and Cosmos-Predict2-2B-Video2World (image+text-to-video).

See the experiment's README for quickstart, guide, and current results.
